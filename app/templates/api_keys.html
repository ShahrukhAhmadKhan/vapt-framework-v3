Why API Keys?
Without API keys: subfinder uses only passive/free sources (crt.sh, wayback, DNS brute) — still finds many subdomains.
With API keys: unlocks Shodan, Chaos, VirusTotal, SecurityTrails, GitHub, and 15+ more sources — finds 10-100x more subdomains.
All keys are stored locally in config/api_keys.json and automatically written to subfinder's provider config.
Add API Key
Keys are stored locally only — never sent anywhere except to the API service during scanning.
Quick Setup — Free Keys
🔵 Shodan Student Account
1. Go to shodan.io → Sign up with your .ac.uk or university email
2. Apply for academic access (free upgrade)
3. Go to Account → API Key → copy it
🟢 VirusTotal (Free)
1. virustotal.com → Sign up free
2. Go to profile → API Key → copy it
🐙 GitHub (Free)
1. github.com → Settings → Developer settings
2. Personal access tokens → Generate new token
3. Select: read:packages only
4. Copy token
⚡ WhoisXML API (Free 500/mo)
1. whoisxmlapi.com → Free signup
2. Dashboard → API Keys → copy it
